# SquirtleChat PRD

## 产品定位
仿微信即时通讯系统，企业级分布式架构，前后端分离。

## MVP 功能
1. 用户注册/登录/登出
2. 好友申请/接受/拒绝/列表/删除
3. 单聊（文本/图片）
4. 群聊（创建/邀请/退群/群消息）
5. WebSocket 实时推送 + 离线消息
6. 文件上传（图片/文件）
7. 多端同步（device_id + sync_seq）
8. 未读数/已读

## 非 MVP（后续）
- 音视频通话、朋友圈、端到端加密、ES 搜索

## 用户角色
- 普通用户
- 群管理员/群主

## 验收标准（全局）
- 两用户单聊实时互通
- 离线后上线 sync 收到消息
- 同账号两设备消息一致
- 群消息所有成员收到
- 文件可上传并在聊天展示
