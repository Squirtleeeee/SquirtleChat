# Phase 101 完成 — 群号与面对面建群规则修正

## 需求分析

- **群号**：约 10 位数字，用于搜索群聊、申请加入
- **4 位数字**：仅用于面对面建群，与群号分离
- **搜索群聊**：按群名称或群号发现群，查看信息后申请加入
- **面对面建群**：独立入口；群主输入 4 位码创建，群员输入相同 4 位码直接加入

## 交付

- `006_group_no_ten.sql` — 群号字段扩为 VARCHAR(16)，清除旧 4 位群号
- 群号生成改为 10 位随机数字
- `GET /groups/discover` — 按群名或群号公开搜索
- 面对面：`POST /face-to-face/start { code }` 群主自定 4 位码；`join` 直接入群
- 前端：`AddContactModal` 四标签（好友/搜索群聊/创建/面对面）；群详情移除建群码区块

## 测试

`go build`、`npm run build`、冒烟测试更新并通过。
