# Phase 47 完成

## 需求分析
- 窄屏下侧栏+主区上下堆叠，聊天区过矮
- 需要抽屉式会话列表：选会话后全屏聊天，可返回列表

## 设计计划
- `≤768px`：侧栏 fixed 抽屉；无会话时全屏列表；有会话时默认隐藏侧栏
- 聊天头「☰」打开列表；遮罩 / Esc 关闭
- 选中好友/群聊后自动收起侧栏

## 交付
- `frontend/src/views/ChatView.vue`：移动端抽屉布局

## 测试

```powershell
cd frontend; npm run build
```

## 验证
- 窄屏无会话：仅侧栏列表
- 点开会话：全屏聊天，☰ 可再开列表
- 点遮罩或 Esc 关闭抽屉
