# Phase 60 完成

## 需求分析
- 聊天主界面存在多处「突然出现/跳一下」：加载更早消息滚顶、上传进度反复吸底、面板无过渡、移动端遮罩瞬显
- 目标：交互节奏接近成熟 IM，减少卡顿与布局跳动

## 设计计划
- 前端-only：滚动锚点、缩小滚动触发条件、composer 叠层 Transition、FAB/遮罩/菜单动效
- 不改 API / 库表

## 交付
- `frontend/src/views/ChatView.vue`
  - 加载更早消息保持滚动位置
  - 消息 watch 改为签名比对（忽略 uploadProgress 抖动）
  - 「回到底部」平滑滚动；草稿切换后自动 resize
  - 搜索/重连/公告/回复/表情/提及/多选/上传条：`<Transition>`
  - 移动端遮罩 fade；右键菜单 clamp + fade-scale
  - 触控下消息操作按钮可点；有气泡上传进度时隐藏底部重复条
  - 缓存会话打开不再闪骨架
- `frontend/src/stores/chat.ts`：搜索高亮时长对齐淡出动画
